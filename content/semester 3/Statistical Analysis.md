# Limits
- one-way causal thinking is an oversimplification
	- multicausality
		- phenomenon are caused by more than one factor
	- bidirectional influence
		- both causal parts are interrelated, effecting each other
		- x -> y; y -> x; x -> y; y -> x; ...
